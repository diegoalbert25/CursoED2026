# Estructura de Datos - EDD-D1

## Descripción
Repositorio de la clase de Estructura de Datos (EDD-D1). Contiene ejemplos, ejercicios y proyectos relacionados con estructuras de datos fundamentales.

## Contenido
- **Listas** - Implementaciones de listas enlazadas y dinámicas
- **Pilas** - Ejemplos y aplicaciones de pilas (LIFO)
- **Colas** - Implementaciones de colas (FIFO)
- **Árboles** - Árboles binarios, recorridos y búsqueda
- **Grafos** - Representaciones y algoritmos básicos
- **Ejercicios** - Problemas propuestos y soluciones

## Lenguajes
- C

## Cómo usar
1. Clonar el repositorio
2. Navegar a la carpeta del tema deseado
3. Compilar/ejecutar según el lenguaje

## Requisitos
- GCC para compilar código en C (MinGW para Windows)
- Git para clonar el repositorio
- GitHub Desktop
- Visual Studio Code
