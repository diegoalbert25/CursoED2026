El presente taller corresponde a la Unidad 5: Métodos de organización eficientes y búsqueda 
binaria. El ejercicio se desarrollará en lenguaje C, utilizando arreglos, funciones, ordenamiento 
eficiente y búsqueda binaria.
Contexto del problema
Una de las terminales portuaria requiere organizar los contenedores ubicados en un patio de 
carga. El patio puede manejar como máximo 30 contenedores. Cada contenedor tiene asociado un 
peso registrado en kilogramos.
El sistema debe permitir:
- Registrar los pesos de los contenedores
- Generar pesos de manera aleatoria
- Mostrar el estado actual del arreglo
- Ordenar los contenedores mediante cada uno de los métodos de ordenamiento eficiente
abordados en la Unidad 5.
- Realizar búsquedas binarias sobre el arreglo de pesos.
